# TaskNote

할 일을 프로젝트별로 정리하고, 캘린더에서 한눈에 확인할 수 있는 Windows 앱입니다.

---

## 주요 기능

- **프로젝트 관리** - 프로젝트별 업무를 색상별로 구분하고, 하위 작업을 자유롭게 추가할 수 있습니다
- **캘린더** - 월간 뷰에서 날짜별 할 일과 일정을 관리할 수 있습니다.
- **정기 업무** - 매주 또는 매월 반복되는 작업을 관리할 수 있습니다
- **위젯 모드** - 작은 창으로 오늘 할 일이나 캘린더를 위젯으로 활용할 수 있습니다.
- **Google Calendar 연동** - 구글 일정과 양방향 동기화되어 일정에 대한 알림을 핸드폰으로 받을 수 있습니다.
- **공유폴더를 통한 멀티데스크 연동** - 데이터를 저장하는 파일의 위치를 공유폴더(구글드라이브, 원드라이브 등)로 설정하여. 업무를 2개이상의 컴퓨터에서 관리할 수 있습니다.
- **다크/라이트 테마** - 두가지 색상테마 설정
- **자동 업데이트** - 새 버전이 나오면 별도로 찾지 않아도 앱에서 확인할 수 있습니다.

---

## 설치

[최신 버전 다운로드](https://github.com/Skedit99/TaskNote/releases/latest)에서 `TaskNote-Setup-x.x.x.exe`를 받아 실행하세요.

### Windows 보안 경고에 대한 안내

아직 코드 서명 인증서가 없어서, 처음 실행할 때 SmartScreen 경고가 나올 수 있습니다.

1. "Windows의 PC 보호" 창이 뜨면
2. **"추가 정보"** 클릭
3. **"실행"** 클릭

서명되지 않은 앱에 공통으로 뜨는 경고입니다. 위 절차를 통해 설치하시면 됩니다.
