# TaskNote

할 일을 프로젝트별로 정리하고, 캘린더에서 한눈에 확인할 수 있는 Windows 데스크톱 앱입니다.

Google Calendar과 연동하면 기존 일정과 함께 관리할 수 있습니다.

---

## 이런 분들에게 추천합니다

- 할 일이 많아서 프로젝트별로 나눠서 관리하고 싶은 분
- 캘린더에 할 일을 배치해서 계획을 세우고 싶은 분
- 바탕화면에 오늘 할 일 위젯을 띄워두고 싶은 분
- Google Calendar을 이미 쓰고 있어서 연동하고 싶은 분

---

## 주요 기능

- **프로젝트 관리** - 색상별로 구분하고, 하위 작업을 무한으로 추가할 수 있습니다
- **캘린더** - 월간 뷰에서 날짜별 할 일과 일정을 확인합니다
- **정기 업무** - 매주 또는 매월 반복되는 작업을 자동으로 생성합니다
- **미니 위젯** - 작은 창으로 오늘 할 일이나 캘린더만 띄워둘 수 있습니다
- **Google Calendar 연동** - 구글 일정과 양방향 동기화됩니다
- **다크/라이트 테마** - 취향에 맞게 전환할 수 있습니다
- **자동 업데이트** - 새 버전이 나오면 앱에서 알려줍니다

## 설치

[최신 버전 다운로드](https://github.com/Skedit99/TaskNote/releases/latest)에서 `TaskNote-Setup-x.x.x.exe`를 받아 실행하세요.

### Windows 보안 경고에 대한 안내

아직 코드 서명 인증서가 없어서, 처음 실행할 때 SmartScreen 경고가 나올 수 있습니다.

1. "Windows의 PC 보호" 창이 뜨면
2. **"추가 정보"** 클릭
3. **"실행"** 클릭

서명되지 않은 앱에 공통으로 뜨는 경고입니다. 소스 코드가 전부 공개되어 있으니 직접 확인하셔도 됩니다.

---

## 데이터는 어디에 저장되나요?

모든 데이터는 **내 PC에만 저장**됩니다. 외부 서버로 전송되지 않습니다.

Google Calendar 연동 시에도 내 기기와 Google 서버 사이에서만 통신하며, 중간에 다른 서버를 거치지 않습니다.
