TaskNote 개인정보처리방침

최종 수정일: 2026년 3월 20일

TaskNote는 사용자의 개인정보를 외부 서버로 전송하지 않습니다. 모든 데이터는 사용자의 PC에만 저장되며, Google Calendar 연동 시 사용자의 Google 계정과 직접 통신합니다.

1. 수집하는 개인정보

TaskNote는 Google Calendar 연동 기능 사용 시 다음 정보를 수집합니다.

수집 항목수집 목적필수 여부
Google 이메일 주소연동된 계정 식별 및 표시선택 (Google 연동 시)
Google 프로필 이름설정 화면에 계정 정보 표시선택 (Google 연동 시)
Google 프로필 사진 URL설정 화면에 계정 정보 표시선택 (Google 연동 시)
Google Calendar 일정 데이터앱과 Google Calendar 간 일정 동기화선택 (Google 연동 시)

Google Calendar 연동은 선택 사항이며, 연동하지 않아도 앱의 기본 기능(일정 관리, 할 일 관리 등)은 정상적으로 사용할 수 있습니다.

2. 개인정보의 저장 및 처리

TaskNote는 별도의 외부 서버를 운영하지 않으며, 모든 데이터는 사용자의 PC 로컬 저장소에만 보관됩니다.

저장 데이터저장 위치형식
일정, 프로젝트, 할 일 등 앱 데이터브라우저 localStorageJSON
Google OAuth 인증 토큰앱 데이터 폴더 (gcal-tokens.json)JSON
Google Calendar 동기화 매핑앱 데이터 폴더 (gcal-mapping.json)JSON

앱 데이터 폴더 위치: %APPDATA%\TaskNote\ (Windows 기준)

3. 제3자 제공

TaskNote는 수집한 개인정보를 제3자에게 제공하지 않습니다.

Google Calendar 연동 시, 사용자의 기기에서 Google 서버로 직접 API 통신이 이루어지며, TaskNote의 별도 서버를 경유하지 않습니다.

4. 개인정보의 보유 및 파기

  • Google 연결 해제 시: OAuth 인증 토큰이 Google 서버에서 폐기되고, 로컬 토큰 파일이 즉시 삭제됩니다.
  • 앱 삭제 시: 앱 데이터 폴더 및 localStorage의 모든 데이터가 삭제됩니다.
  • 별도의 보유 기간 없이, 사용자가 삭제 조치를 취하면 즉시 파기됩니다.

5. 이용자의 권리

사용자는 언제든지 다음 조치를 취할 수 있습니다.

  • Google 연결 해제: 설정 > Google 캘린더 > "연결 해제" 버튼
  • Google 계정에서 앱 접근 권한 제거: Google 계정 > 보안 > 서드파티 앱에서 TaskNote 접근 권한 삭제
  • 로컬 데이터 삭제: 앱 삭제 또는 설정 > "데이터 초기화" 기능 사용

6. Google API 서비스 이용

TaskNote는 Google API 서비스를 사용하며, Google API 서비스 사용자 데이터 정책(제한적 사용 요건 포함)을 준수합니다.

TaskNote는 Google API를 통해 얻은 데이터를 다음 목적으로만 사용합니다.

  • 사용자의 Google Calendar 일정을 앱에 표시하고 동기화
  • 사용자의 계정 정보(이메일, 이름)를 설정 화면에 표시

7. 보안

  • Google OAuth 2.0 인증에 PKCE(Proof Key for Code Exchange) 방식을 사용합니다.
  • Electron 앱의 Context Isolation 및 Node Integration 비활성화를 통해 보안을 강화합니다.
  • 인증 토큰은 사용자의 로컬 PC에만 저장되며 암호화된 HTTPS를 통해 Google 서버와 통신합니다.

8. 아동의 개인정보

TaskNote는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 개인정보처리방침의 변경

본 방침이 변경되는 경우 앱 업데이트 및 본 페이지를 통해 공지합니다.
