프로젝트 관리

프로젝트와 세부 업무를
체계적으로 관리

색상 태그로 프로젝트를 구분하고, 세부 업무를 추가하여 진행 상황을 한눈에 파악하세요. D-Day 카운터로 마감일도 놓치지 않습니다.

  • 8가지 프로젝트 색상 태그
  • 세부 업무 추가 및 완료 체크
  • D-Day 자동 카운터
업무 편집

업무 시간과 메모를
간편하게 설정

시작 시간과 종료 시간을 설정하고, 메모를 추가하여 업무를 더욱 구체적으로 관리하세요. 설정한 시간은 Google 캘린더에도 자동 반영됩니다.

  • 시작·종료 시간 설정
  • 메모 및 설명 추가
  • Google 캘린더 자동 반영
정기 업무

반복 업무를 한 번만
등록하면 끝

매주 회의, 매월 보고서 등 정기적인 업무를 자동으로 캘린더에 등록합니다. 격주, 3주, 4주 간격까지 다양한 패턴을 지원합니다.

  • 주간·월간 반복 설정
  • 격주·3주·4주 간격 지원
  • 시작일·종료일 지정 가능
위젯 모드

바탕화면에서 바로
오늘의 할 일 확인

컴팩트한 위젯을 바탕화면에 고정하세요. 오늘 할 일 목록, 캘린더 뷰, 축소형 위젯 등 다양한 모드를 지원합니다. 항상 위/아래 모드로 작업 중에도 확인 가능합니다.

  • 오늘 할 일 위젯
  • 캘린더 위젯 (공휴일 표시)
  • 항상 위에·항상 아래 모드
Google Calendar

Google 캘린더와
완벽하게 연동

앱에서 추가한 업무가 Google 캘린더에 자동 반영되고, Google 캘린더의 일정도 앱에 표시됩니다. 완료 상태까지 실시간 동기화됩니다.

  • 양방향 실시간 동기화
  • 공휴일 자동 표시
  • 오프라인 변경사항 자동 동기화
