TaskNote

프로젝트 업무, 독립 일정, 정기 업무를 한눈에 관리하는 Windows 데스크톱 앱

다운로드 (Windows)

주요 기능

프로젝트 관리

  • 프로젝트별 세부 업무를 트리 구조로 관리
  • 색상 태그로 프로젝트 구분
  • 드래그 앤 드롭으로 업무 순서 변경

캘린더

  • 월간 캘린더에서 업무 일정을 한눈에 확인
  • 날짜별 업무 예약 및 관리
  • 공휴일 자동 표시

오늘 할 일

  • 오늘의 업무를 우선순위로 정리
  • 완료 체크 및 진행률 확인
  • 위젯 모드로 바탕화면에서 바로 확인

정기 업무

  • 매주/매월 반복되는 업무 자동 등록
  • 격주, 특정 요일 등 다양한 반복 패턴 지원
  • 시작일/종료일 설정

Google 캘린더 연동

  • 양방향 실시간 동기화
  • 앱에서 추가한 업무가 Google 캘린더에 자동 반영
  • Google 캘린더에서 추가한 일정도 앱에 자동 등록

위젯 모드

  • 컴팩트한 위젯으로 바탕화면에 고정
  • 오늘 할 일 목록과 진행률 표시
  • 항상 위 모드 지원

기술 스택
